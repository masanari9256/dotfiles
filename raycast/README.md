# インストール方法
1. Raycastで「command + ,」を押して設定を開きます
2. 「Advanced（詳細設定）」タブをクリックします
3. Raycastの設定をエクスポートする場合：
   - 「Export（エクスポート）」ボタンをクリックします
   - `~/dotfiles/raycast` ディレクトリを選択します
4. Raycastの設定をインポートする場合：
   - 「Import（インポート）」ボタンをクリックします
   - 「Select file（ファイルを選択）」ボタンをクリックします
   - `~/dotfiles/raycast/Raycast yyyy-mm-dd hh.mm.ss.rayconfig` ファイルを選択します
   - 「Import（インポート）」ボタンをクリックします
   - 確認画面で「Import（インポート）」ボタンをクリックします
   - 「Continue（続行）」ボタンをクリックして完了します
